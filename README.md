# Portfolio Website

This is a simple, modern personal portfolio website built with HTML, CSS, and JavaScript. It features sections for About, Projects, and Contact, and is fully responsive and easy to customize.

## Features
- Clean and modern design
- Responsive layout for desktop and mobile
- About section for personal introduction
- Projects section to showcase your work
- Contact form with client-side validation

## File Structure
- `index.html` — Main HTML file containing the structure of the portfolio
- `styles.css` — CSS file for styling and layout
- `script.js` — JavaScript for contact form validation and interactivity

## Usage
1. Clone or download this repository to your local machine.
2. Open `index.html` in your web browser to view your portfolio.
3. Edit the content in `index.html` to personalize your name, about info, and projects.
4. Customize styles in `styles.css` as desired.
5. The contact form is functional on the client side; for real submissions, connect it to a backend or email service.

## Customization
- Change the name, about text, and project details in `index.html`.
- Add more projects by duplicating the project div in the Projects section.
- Update colors, fonts, and layout in `styles.css`.
